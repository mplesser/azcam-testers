Report Commands¶

The report commands are helpful when generating reports.

Usage Example:

from azcam_testers import report
report.make_rstfile("rstfile.rst")

Report generation class.

class Report¶

Report generation methods.

make_mdfile(md_file, lines=[])¶

Create a markdown file from lines.

make_rstfile(rst_file, lines=[])¶

Create an RST file from lines.

md2pdf(filename, options=[], create_html=False)¶

Create a PDF from a markdown file. Filename extension is optional. Optionally also create an html file.

merge_pdf(input_files=[], output_file='combined.pdf')¶

Merge multiple PDF files into one.

rst2pdf(filename, options=[])¶

Create a PDF from an RST file. Filename extension is optional. Also create an html file (currently required).
